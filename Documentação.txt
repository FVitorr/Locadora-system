Documentação referente ao projeto da locadora:

------------------------------ Padrão de Projeto: -----------------------------------

	Nome dos arquivos em letra minusculas.
	
	Arquivos .h devem conter mesmo nome do arquivo ao qual aponta
	
	Bases de dados devem ser carregadas:        Quando:                                             Init Dev.
			
			-> Locadora*   						Inicializar e Alterar.								Brenin
			-> Filmes							Inicializar e Alterar.								EL BRABO
			-> Categorias						Alterar e Listar.									EL BRABO
			-> Cliente							Inicializar, Alterar, Processo de Emprestimo.		Brenin
	
	
			*Dados de endereço encontra-se aqui {
			
				Talvez Seja viavel a criação de uma função que possa Alterar o endereço.
				
			}
			
			
	Variaveis Globais:
	
		Referencia aos Dados em Memoria e VARIAVEIS GLOBAIS :
		
			Utilizar caracteres MAIUSCULO ao definir o nome das variavel:
			
				fCategoria *BD_CAT; -> Definição de Array de fCategoria Dinamico
	
				int QTD_CAT = 0, -> Quantida de fCategoria no Array
				
				TAM_CAT = 1; -> Tamanho do Array de fCategoria ao iniciar;

			------------------------------------------ Estruturas de TEMPO ------------------------------------------
			
			Entradas de dados referente ao tempo seram executadas pelas seguintes funçoes
			
			typedef struct hora; \\ Objeto Hora 
			
				Formato: hh:mm:ss
			
			void obterHour(hora *n) \\ Obter hora atual do SISTEMA , entrada ponteiro do tipo hora.
			
			typedef struct data; \\Objeto Data.
			
				Formato: dd/mm/yy
			
			void obterData(data *n) \\ Obter data atual do SISTEMA, entrada ponteiro do tipo data.
